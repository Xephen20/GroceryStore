# Grocery Store application 
Install dependencies
```sh
apt install python3-pip
pip3 install --upgrade pip
pip3 install virtualenv
```
Build venv
```sh
source ./enviroment/project_teldat/bin/activate
pip3 install --upgrade pip
pip3 install -r ./enviroment/requirements.txt
```
Run Script
```sh
python3 main.py
```
### Instruction
This program is made for adding products to cart and print / send recipe.
1. Open program.
2. For add procut, please choose "Add item to cart".
3. Choose Product Category, Item, Variety, and Weight.
4. Product was added to the cart.
5. You can add multiple products and check receip in main menu.
6. In finish you can send receip to the server or print all data with final price.
7. After close program you can delete all items from receip or add more items for existing.
