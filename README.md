# Grocery Store application 
Install dependencies
```sh
apt install python3-pip
pip3 install --upgrade pip
pip3 install virtualenv
```
Build venv
```sh
virtualenv ./enviroment/project_teldat
pip3 install --upgrade pip
pip3 install -r ./enviroment/requirements.txt
```
Run Script
```sh
python3 main.py
```
